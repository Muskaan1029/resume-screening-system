- Simple Flask web interface
- Professional output with colored results

- ## Dataset

The project uses a resume dataset sourced from Kaggle. 
It contains columns: `Resume_str` (resume text) and `Category` (HR or Designer). 
The dataset was used to train the machine learning model. 
Due to size limitations, the dataset is not included in this repository.
